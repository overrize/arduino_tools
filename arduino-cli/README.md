# 🎯 Arduino 开发 CLI 工具

友好的命令行工具，帮助你快速开始 Arduino 开发。

## 🎨 版本选择

我们提供三个版本，满足不同需求：

| 版本 | 风格 | 特点 | 适用场景 |
|------|------|------|----------|
| **V1** | 传统菜单 | 引导式、结构化 | 初学者、完整配置 |
| **V2** | 简洁命令行 | 快速、直接 | 快速原型、熟练用户 |
| **V3** | BootLoop 风格 | 炫酷、交互式 | 演示、体验优先 |

### 快速选择

- 🆕 **第一次使用？** → 选择 V1
- ⚡ **追求效率？** → 选择 V2
- 🎨 **追求视觉？** → 选择 V3

详细对比：[docs/cli-tool/VERSION_COMPARISON.md](../docs/cli-tool/VERSION_COMPARISON.md)

## 📦 安装

### 1. 克隆或下载代码

```bash
cd arduino-cli
```

### 2. 安装依赖（可选）

```bash
pip install -r requirements.txt
```

> 注：pyperclip 用于剪贴板功能，如果不安装也能正常使用

---

## 🚀 快速开始

### V3 - BootLoop 风格（推荐）

```bash
# 交互模式
arduino-dev-v3.bat

# 直接模式
arduino-dev-v3.bat 用 Pico 做一个 LED 闪烁
```

### V2 - 简洁命令行

```bash
# 交互模式
arduino-dev-v2.bat

# 直接模式
arduino-dev-v2.bat 用 Pico 做一个 LED 闪烁
```

### V1 - 传统菜单

```bash
arduino-dev.bat
```

## 📋 功能

### 1. 新建项目

引导式创建 Arduino 项目：

1. 选择板型（Pico / Uno / Nano / ESP32）
2. 选择项目类型（LED 闪烁 / 按钮控制 / 传感器）
3. 配置参数（引脚、间隔等）
4. 自动生成 Kiro 命令
5. 打开 Kiro 并粘贴命令

### 2. 编译项目（即将推出）

编译 Arduino 代码

### 3. 上传项目（即将推出）

上传代码到硬件

### 4. 监控串口（即将推出）

查看串口输出

### 5. 帮助

显示使用说明

## 📚 工作流程

```
CLI 工具
  ↓
引导式收集需求
  ↓
生成 Kiro 命令
  ↓
复制到剪贴板
  ↓
打开 Kiro
  ↓
用户粘贴命令
  ↓
Kiro 调用 MCP Server
  ↓
生成代码 + Wokwi 文件
  ↓
完成！
```
