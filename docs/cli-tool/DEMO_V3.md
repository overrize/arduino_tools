## 🎥 录制建议

### 准备工作

1. **终端设置**
   - 使用 Windows Terminal
   - 字体：Cascadia Code, 14pt
   - 主题：Dark+
   - 透明度：90%

2. **窗口大小**
   - 宽度：120 列
   - 高度：30 行
   - 全屏或居中

3. **录制工具**
   - OBS Studio
   - ScreenToGif
   - Windows Game Bar

### 录制脚本

**时长**：2-3 分钟

**步骤**：

1. **开场（10s）**
   - 显示桌面
   - 打开终端
   - 运行 `arduino-dev-v3.bat`

2. **展示界面（10s）**
   - 停留在欢迎界面
   - 展示 ASCII 标题
   - 展示示例

3. **创建项目（30s）**
   - 输入：`用 Pico 做一个 LED 闪烁，25 号引脚`
   - 展示加载动画
   - 展示进度提示
   - 展示 Kiro 启动

4. **使用命令（20s）**
   - 输入：`/help`
   - 展示帮助信息
   - 输入：`/clear`
   - 展示清屏效果

5. **连续操作（40s）**
   - 创建第二个项目
   - 展示连续工作流
   - 展示智能提示

6. **结束（10s）**
   - 输入：`/exit`
   - 展示退出信息
   - 关闭终端
