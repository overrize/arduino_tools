# Arduino Dev CLI 版本对比

## 版本概览

| 版本 | 风格 | 特点 | 适用场景 |
|------|------|------|----------|
| V1 | 传统菜单 | 结构化、引导式 | 初学者、完整配置 |
| V2 | 简洁命令行 | 快速、直接 | 快速原型、熟练用户 |
| V3 | BootLoop 风格 | 炫酷、交互式 | 演示、体验优先 |

## 详细对比

### V1 - 传统菜单式

**界面风格**：
```
============================================================
🎯 Arduino 开发助手
============================================================

欢迎使用 Arduino 自然语言开发工具！

请选择操作：
  1. 创建新项目
  2. 查看示例
  3. 配置设置
  4. 退出

请输入选项 (1-4): 
```

**特点**：
- ✅ 结构化菜单
- ✅ 引导式操作
- ✅ 完整的配置选项
- ✅ 适合初学者
- ❌ 操作步骤多
- ❌ 界面较传统

**使用场景**：
- 第一次使用
- 需要完整配置
- 不熟悉命令行

**启动方式**：
```bash
arduino-dev.bat
```

---

### V2 - 简洁命令行

**界面风格**：
```
============================================================
🎯 Arduino 开发助手
============================================================

欢迎使用 Arduino 自然语言开发工具！

💡 示例：
  • 用 Pico 做一个 LED 闪烁，25 号引脚
  • 用 Uno 做一个 LED 闪烁，13 号引脚，每 2 秒闪一次

请描述你想做什么：
> 
```

**特点**：
- ✅ 简洁直接
- ✅ 快速输入
- ✅ 支持命令行参数
- ✅ 适合熟练用户
- ❌ 界面较简单
- ❌ 缺少视觉反馈

**使用场景**：
- 快速原型开发
- 熟悉工具流程
- 脚本自动化

**启动方式**：
```bash
# 交互模式
arduino-dev-v2.bat

# 直接模式
arduino-dev-v2.bat 用 Pico 做一个 LED 闪烁
```

---

### V3 - BootLoop 风格

**界面风格**：
```
    ___    ____  ____  __  ______   ______     ___    ______
   /   |  / __ \/ __ \/ / / /  _/  / ____/    /   |  / ____/
  / /| | / /_/ / / / / / / // /   / / __     / /| | / / __  
 / ___ |/ _, _/ /_/ / /_/ // /   / /_/ /    / ___ |/ /_/ /  
/_/  |_/_/ |_/_____/\____/___/   \____/____/_/  |_|\____/   
                                      /_____/                

Welcome to the Arduino Agent!

Ctrl + C to exit. Type / for commands.

💡 Examples:
  1. 用 Pico 做一个 LED 闪烁，25 号引脚
  2. 用 Uno 做一个 LED 闪烁，13 号引脚，每 2 秒闪一次

▶ 
```

**特点**：
- ✅ ASCII 艺术标题
- ✅ 彩色输出
- ✅ 加载动画
- ✅ 命令系统
- ✅ 视觉反馈丰富
- ✅ 适合演示
- ❌ 需要支持 ANSI 颜色的终端

**使用场景**：
- 产品演示
- 视频录制
- 追求视觉体验
- 交互式探索

**启动方式**：
```bash
# 交互模式
arduino-dev-v3.bat

# 直接模式
arduino-dev-v3.bat 用 Pico 做一个 LED 闪烁
```

## 使用建议

### 选择 V1 如果你：
- 🆕 第一次使用工具
- 📚 需要完整的引导
- ⚙️ 需要配置多个选项
- 🎓 正在学习 Arduino

### 选择 V2 如果你：
- ⚡ 追求速度和效率
- 🔧 熟悉工具流程
- 🤖 需要脚本自动化
- 💻 习惯命令行操作

### 选择 V3 如果你：
- 🎨 追求视觉体验
- 🎬 需要演示或录制
- 🚀 喜欢现代化界面
- 💡 想要交互式探索

| 内存占用 | ~15MB | ~12MB | ~20MB |

## 迁移指南

### 从 V1 迁移到 V2
```bash
# V1
arduino-dev.bat
> 选择 1 (创建新项目)
> 输入需求

# V2
arduino-dev-v2.bat
> 直接输入需求
```

### 从 V2 迁移到 V3
```bash
# V2
arduino-dev-v2.bat 用 Pico 做一个 LED 闪烁

# V3
arduino-dev-v3.bat 用 Pico 做一个 LED 闪烁
# 或交互模式
arduino-dev-v3.bat
▶ 用 Pico 做一个 LED 闪烁
```

- ✅ 示例展示

## 推荐配置

### 最佳体验（V3）
- 终端：Windows Terminal
- 字体：Cascadia Code / Fira Code
- 主题：Dark+
- 编码：UTF-8

### 快速开发（V2）
- 终端：PowerShell
- 快捷方式：桌面快捷方式
- 别名：`alias ard='arduino-dev-v2.bat'`

### 学习使用（V1）
- 终端：CMD / PowerShell
- 文档：README.md
- 示例：examples/
