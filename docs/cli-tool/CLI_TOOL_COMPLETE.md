## 🚀 如何使用

### 快速开始

```bash
# 1. 进入目录
cd arduino-cli

# 2. 运行工具
arduino-dev.bat

# 或
python arduino_dev.py
```

### 完整流程

```
1. 启动 CLI 工具
   ↓
2. 选择"新建项目"
   ↓
3. 选择板型（Pico/Uno/Nano/ESP32）
   ↓
4. 选择项目类型（LED 闪烁等）
   ↓
5. 输入参数（引脚、间隔、名称）
   ↓
6. 确认创建
   ↓
7. 命令自动生成并复制到剪贴板
   ↓
8. Kiro 自动打开（如果找到）
   ↓
9. 在 Kiro 中粘贴命令（Ctrl+V）
   ↓
10. Kiro 调用 MCP Server
   ↓
11. 代码和 Wokwi 文件生成
   ↓
12. 完成！
```

## 🎯 解决的问题

### 问题 1：MCP Server 需要明确提及
**之前**：用户必须说"使用 Arduino MCP server 的 xxx 工具"  
**现在**：CLI 工具自动生成正确的命令 ✅

### 问题 2：新手不知道如何开始
**之前**：不知道有哪些工具，不知道命令格式  
**现在**：引导式界面，一步步收集需求 ✅

### 问题 3：命令格式复杂
**之前**：需要记住工具名称和参数格式  
**现在**：自动生成，自动复制到剪贴板 ✅

### 问题 4：缺少入口
**之前**：直接在 Kiro 中输入，容易出错  
**现在**：专门的 CLI 入口，体验流畅 ✅
