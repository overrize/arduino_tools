  Serial.begin(9600);
  Serial.println("LED Blink Started");
  Serial.println("Pin: 25");
  Serial.println("Interval: 1000ms");
}

void loop() {
  digitalWrite(LED_PIN, HIGH);
  Serial.println("LED ON");
  delay(1000);
  
  digitalWrite(LED_PIN, LOW);
  Serial.println("LED OFF");
  delay(1000);
}

## 解决的技术问题

### 问题 1: arduino-cli 路径问题
**症状**: `arduino-cli: The term 'arduino-cli' is not recognized`

**解决方案**:
1. 添加到系统 PATH
2. 代码中自动检测常见安装位置

**实现**:
```python
possible_paths = [
    "arduino-cli",
    "F:\\Arduino\\arduino-cli.exe",
    "%LOCALAPPDATA%\\Arduino15\\arduino-cli.exe",
    "C:\\Program Files\\Arduino CLI\\arduino-cli.exe",
]
```

### 问题 2: JSON 格式兼容性
**症状**: `'str' object has no attribute 'get'`

**原因**: arduino-cli 1.x 的 JSON 格式与 0.x 不同

**解决方案**: 同时支持新旧格式
```python
# Handle new JSON format (arduino-cli 1.x)
if "detected_ports" in data:
    for item in data["detected_ports"]:
        port_info = item.get("port", {})
        # ...
# Handle old JSON format (arduino-cli 0.x)
else:
    for item in data:
        # ...
```

### 问题 3: Pico BOOTSEL 模式
**理解**: Pico 在 BOOTSEL 模式下显示为 USB 大容量存储设备

**处理**: arduino-cli 自动处理 UF2 上传
