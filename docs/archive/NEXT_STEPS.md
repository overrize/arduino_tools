## 📋 下一步开发优先级

### 🔥 优先级 1: 增强调试体验（1-2 天）

**目标**：让用户能够实时看到程序运行状态

#### 1.1 实时串口监控
**功能**：
- [ ] 在 MCP 工具中集成实时串口监控
- [ ] 支持自动重连
- [ ] 彩色输出（错误红色、警告黄色）
- [ ] 时间戳显示

**实现**：
```python
Tool(
    name="monitor_serial_realtime",
    description="Real-time serial monitoring with auto-reconnect"
)
```

**用户体验**：
```
用户: "监控 Pico 的串口输出"
系统: 实时显示串口数据，带时间戳和颜色
```

#### 1.2 串口数据过滤
**功能**：
- [ ] 按关键词过滤
- [ ] 正则表达式搜索
- [ ] 数据统计（错误数、警告数）

**示例**：
```
用户: "只显示包含 'ERROR' 的输出"
系统: 过滤并高亮显示错误信息
```

#### 1.3 串口绘图器
**功能**：
- [ ] 解析数值数据
- [ ] 实时绘制曲线
- [ ] 支持多通道

**示例**：
```cpp
Serial.println("temp:25.5");
Serial.println("humidity:60.2");
```
→ 自动绘制温湿度曲线

```python
# 在 templates.py 中添加
BUTTON_LED_TEMPLATE = """
const int BUTTON_PIN = {{ button_pin }};
const int LED_PIN = {{ led_pin }};

void setup() {
  pinMode(BUTTON_PIN, INPUT_PULLUP);
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  if (digitalRead(BUTTON_PIN) == LOW) {
    digitalWrite(LED_PIN, HIGH);
  } else {
    digitalWrite(LED_PIN, LOW);
  }
  delay(50);
}
"""
```
